# AniFrame AMV Studio

A clean, cinematic landing page for an anime AMV editing studio. The site presents featured work, service packages, process details, and an order section in a polished single-page layout.

## Open the Website

- [Open the homepage](./index.html)

## Project Overview

- Built with plain HTML, CSS, and JavaScript.
- Main entry point: `index.html`
- Styles: `css/styles.css`
- Scripts: `js/script.js`

## How to View

1. Open `index.html` in a browser.
2. For GitHub, enable GitHub Pages from the repository settings if you want a public live link.

## Preview

This site is designed for a bold anime-styled presentation with a focused call to action for editing services.
